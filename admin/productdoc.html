Online FishCart Help

Context Sensitive Help

If you have a Javascript enabled browser, just click into the field that you need help on, then roll your mouse over the Help button; a context sensitive screen will pop up automatically for that field.

If your browser does not support Javascript, then just click on the Help button and find the particular help section you need.

Short Product Description

  • The short description is used when viewing the shopping cart contents; it can contain up to 255 characters. It can contain HTML markup, such as <font>, <br>, etc. The text should NOT be wrapped with quote signs ("...").

Product Description

  • The product description can contain up to 255 characters; it can contain HTML markup, such as <font>, <br>, etc. The text should NOT be wrapped with quote signs ("...").

Product Keywords

  • The product keywords field can contain up to total 255 characters. All searches against keywords the buyer enters are done against the keywords in this field. This can help provide alternate ways for clients to find products by other than listed categories.
  • Enter all desired keywords as space separated words; no commas or quotes are necessary.

Audio / Video Clip URI

  • Enter the absolute path location from your Web directory root to an audio or video file for this product.
  • Any type of Internet audio or video can be used in this link.
  • Leave blank if no audio or video clip is available for this product.
  • Example: /sound/soundbyte.ram

Product Image URI

  • Enter the absolute path location from your Web directory root to the product graphic.
  • Example: /images/prodpic.gif

Product Thumbnail Image URI

  • Enter the absolute path location from your Web directory root to the product thumbnail graphic.
  • Example: /images/prodthumb.gif
  • The thumbnail graphic is optionally used:
    if a product is specified in a promotional slot and
    if the top banner graphic in the Web table and the product banner are both undefined.
  • This field is optional.

Product Banner Image URI

  • Enter the absolute path location from your Web directory root to the product banner graphic.
  • Example: /images/prodbanner.gif
  • The product banner graphic is optionally used:
    if a product is specified in a promotional slot and
    if the top banner graphic in the Web table is undefined.
  • This field is optional.

Graphic Width/Height

  • Enter the width and height of the graphic in pixels.
  • Optional; if not specified, the values will be set to those of the specified graphic.

Product SKU

  • The format of the SKU is dependent on each client's particular inventory identification scheme. Enter the SKU as required, respecting capital letter and numbers as your inventory control requires.

Product Price

  • Enter the product price as below. An example of a valid price would be 1345.00. This price will be used for cart calculations, so it must be entered exactly as required.
    • with exactly 2 decimal places
    • without a leading currency sign
    • without commas

Product Inventory Quantity

  • Optional; check the enable inventory tracking checkbox to track inventory balance.
  • Order quantities will not be allowed to exceed outstanding inventory.
  • Inventory balance is downcounted when the order is placed.
  • When inventory goes to 0, the product will no longer be displayed for sale.

Product Offer # / Motivation Code

  • Optional; enter a product offer number or motivation code for this product. The format and length of this field are vendor dependent; it can contain both numbers and alpha characters.

New Product Category Selection

  • The product will be placed in the selected categories automatically when the product is added to the database. You may add the product to other categories also from the product maintenance page; there is no limit on the number of categories a product can be listed in.

Modify/Delete Product

  • Select a product to modify or delete;
  • Click the appropriate "Modify" or "Delete" button.

Attention Drawing Splash Graphic

  • Enter the absolute path location from your Web directory root to a small accent graphic to draw attention to the item.
  • Example: /images/splash.gif
  • This field is optional.

Sale Price

  • Enter the sale price as below. An example of a valid price would be 1345.00. This price will be used for cart calculations, so it must be entered exactly as required.
    • with exactly 2 decimal places
    • without a leading currency sign
    • without commas

Sale Start/End Dates

  • Enter the start and end date of the sale.
  • If the current date is after the start and before the end date, the item will be shown as on sale.
  • Both date fields must be given.
