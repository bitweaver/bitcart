RGB to Color Name Mapping

This is a composite of MIT's Xconsortum red/green/blue (RGB) color specifications, Xconsortum version 10.41, 1994.

The color table has been roughly divided up into shades. Select one of these: Black; Blue; Brown; Gray; Green; Orange; Red; Violet; White; Yellow; to jump to that division.

The "BG/FG color sample" is made by changing both the background and foreground colors. The background is changed with 


	<... BGCOLOR=#xxxxxx ...>
and foreground is changed with 

	<FONT COLOR=#xxxxxx>
You will see a solid block of color if your browser supports both markups; or just "### SAMPLE ###" in color if just one is supported. (Thanks to Kenji Mikasa for the idea of using BG/FG samples.)
