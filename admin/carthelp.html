FishCart Shopping Cart Help

Context Sensitive Help

If you have a Javascript enabled browser, just click into the field that you need help on, then roll your mouse over the Help button; a context sensitive screen will pop up automatically for that field.

If your browser does not support Javascript, then just click on the Help button and find the particular help section you need.

Product Description

  • The product description can contain up to 255 characters; it can contain HTML markup, such as <font>, <br>, etc. The text should NOT be wrapped with quote signs ("...").

Web Image URIs

  • The URIs for various Web images should be absolute file specifications, such as "/images/sandbanr.gif". The quote signs should NOT be given as part of the specification.

Product Image URI

  • Enter the absolute path location from your Web directory root to the product graphic for the main catalog page.
  • Example: /images/bkground.jpg

Graphic Width/Height

  • The width and height of a graphic are given in pixels; use a graphics program to find the dimensions of each graphic. Specifying the graphic size allows the Web page text to begin displaying more quickly in most browsers.

Product/Category Association

  • Select a category to see all products that are associated with that category. You will be able to add or delete products in the category.
  • There is no limit on the number of categories a given product can be listed in.

Product SKU

  • The format of the SKU is dependent on each client's particular inventory identification scheme. Enter the SKU as required, respecting capital letter and numbers as your inventory control requires.

Product Price

  • Enter the product price as below. An example of a valid price would be 1345.00. This price will be used for cart calculations, so it must be entered exactly as required.
    • with exactly 2 decimal places
    • without a leading currency sign
    • without commas

Product Inventory Quantity

  • This is not currently implemented.

New Product Category Selection

  • The product will be placed in the selected categories automatically when the product is added to the database. You may add the product to other categories also as required; there is no limit on the number of categories a product can be listed in.

Show All Products

  • Put an 'X' in this box to show all products in the modify/delete list.

SKU Search Options

  • Enter a specific SKU to modify or delete.
  • OR, enter a partial SKU pattern and check:
  • "SKU Begins With" to select all SKUs that begin with the pattern in the SKU field.
  • "SKU Containsquot; to select all SKUs that have the SKU pattern anywhere in them.
  • "SKU Ends With" to select all SKUs that end with the pattern in the SKU field.

Web Page Color Hex Values

  • The hexadecimal color values given on Web pages are exactly six valid hex characters, from 0-9, A-F or a-f.
  • These fields can be blank; colors will be the browser default.
  • These fields are case independent.
