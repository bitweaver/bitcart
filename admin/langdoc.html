Language Context Sensitive Help

If you have a Javascript enabled browser, click into the field that you need help on, then roll your mouse over the Help button; a context sensitive screen will pop up automatically for that field.

If your browser does not support Javascript, then click on the Help button and find the particular help section you need.

Language Zone

  • Each language is assigned to a particular zone.
  • Select the desired zone for this language.

Language Profile Description

  • Enter a descriptive title for this language.
  • Examples might be "US English" or "UK English".

Show Cart Script

  • The name of the file to display contents of an order.
  • The present default is showcart.php.

Show Geography Script

  • The name of the file to select the geographic area of the buyer.
  • The present default is showgeo.php.

Order Entry Script

  • This file requests billing, shipping and credit information to place the buyer's order.
  • The present default is orderform.php.

Order Processing Script

  • This file that processs a submitted order.
  • It verifies overall information integrity.
  • Typically only error messages are shown to correct required information.
  • The online or offline modules are called as defined in the Vendor Profile.

Order Final Processing Script

  • This file displays a thank you message after the order processing is complete.
  • The present default is orderfinal.php.
