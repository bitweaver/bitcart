Category Specific Context Sensitive Help

If you have a Javascript enabled browser, click into the field that you need help on, then roll your mouse over the Help button; a context sensitive screen will pop up automatically for that field.

If your browser does not support Javascript, then click on the Help button and find the particular help section you need.

Category Description

  • The category description can contain up to 80 characters.
  • It can contain HTML markup, such as <font>, <br>, etc.
  • The text should NOT be wrapped with quote signs ("...").

Category Logo

  • This is the masthead logo displayed on the results page when products from a particular category are displayed.
  • If this field is left blank, the masthead logo from the web table will be displayed.
  • This field is optional and can be left blank.

Category Logo Dimensions

  • Width and height dimensions of the category masthead logo.
  • Optional; if not specified, the values will be set to those of the specified graphic.

Category Background Graphic

  • The background graphic can be different for each product category. This gives you the flexibility to alter the appearance of the displayed Web pages to complement each category.
  • This field is optional and can be left blank.

Category Background Color

  • The background color is given in the <body ... bgcolor= > attribute. It can be different for each product category. This gives you the flexibility to customize the appearance of the displayed Web pages to best fit each product category.
  • This field is optional and can be left blank.
  • If given, this field must contain exactly 6 hexadecimal characters.

Category Text and Link Colors

  • The text and link colors are given in the <body ... text= link= vlink= alink= > attributes. The text and link colors can be different for each product category. This gives you the flexibility to customize the appearance of the displayed Web pages to best fit each product category.
  • These fields are optional and can be left blank.
  • If given, these fields must contain exactly 6 hexadecimal characters.

Category Promotional Banner

  • This field allows you to specify a banner graphic to be displayed at the top of each page in conjuction with the Category Promotional SKU.
  • See the Category Promotional SKU help section for more information.
  • This field is can be left blank if the Category Promotional SKU is blank.
  • If the Category Promotional SKU field is given, this field must be specified.

Category Promotional Banner Dimensions

  • The width and height of the Category Promotional Banner.
  • Optional; if not specified, the values will be set to those of the specified graphic.

Category Promotional SKU

  • This field allows you to specify the SKU of a product to be displayed at the top of each page displayed for a given category. This might typically be used for special promotions on products.
  • It is not necessary that the product specified be part of the category in which it is displayed.
  • If given, the Category Promotional Banner must be specified also.
  • This field is optional and can be left blank.
