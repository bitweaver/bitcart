Online FishCart Help

Context Sensitive Help

If you have a Javascript enabled browser, just click into the field that you need help on, then roll your mouse over the Help button; a context sensitive screen will pop up automatically for that field.

If your browser does not support Javascript, then just click on the Help button and find the particular help section you need.

Home Page URL

  • Enter the full URL of your home page.
  • Example: http://www.yourdomain.com/

Online Catalog Page URL

  • Enter the full URL of your online catalog page. This is the main page for the shopping cart.
  • Example: http://www.yourdomain.com/resources/

Web Page Title

  • Enter the text that will appear both as the HTML <title> and as a banner across the top of the page.
  • Keep the title short; a few precise words is best.

Background Graphic URI

  • Enter the absolute path location from your Web directory root to the background texture graphic for the main catalog page.
  • Example: /images/bkground.jpg

Main Page Masthead Logo

  • Enter the absolute path location from your Web directory root to the masthead logo graphic for the main catalog page.
  • Example: /images/logo.gif

Graphic Dimensions

  • Enter the dimensions in pixels of the appropriate graphic image.
  • Optional; if not specified, the values will be set to those of the specified graphic.

Web Page Color Hex Values

  • The hexadecimal color values given on Web pages are exactly six valid hex characters, from 0-9, A-F or a-f.
  • These fields can be blank; colors will be the browser default.
  • These fields are case independent.
  • Hexadecimal Color Chart

Header Product SKU

  • Optional; used to specify the product for the header product promotional banner.
  • The given SKU must already exist in the database.

Header Product Banner

  • Not used unless the Header Product SKU is given.
  • If given, overrides the product banner, thumbnail, and picture.
  • Enter the absolute path location from your Web directory root to the banner graphic.
  • Example: /images/headbanner.gif

Header Product Text

  • Optional descriptive text for the header product.
  • Not recommended; a single banner graphic should be used instead.

Footer Product SKU

  • Optional; used to specify the product for the footer product promotional banner.
  • The given SKU must already exist in the database.

Footer Product Banner

  • Not used unless the Footer Product SKU is given.
  • If given, overrides the product banner, thumbnail, and picture.
  • Enter the absolute path location from your Web directory root to the banner graphic.
  • Example: /images/footbanner.gif

Footer Product Text

  • Optional descriptive text for the footer product.
  • Not recommended; a single banner graphic should be used instead.

Category Header Text

  • Optional descriptive text that goes above the category list on the main display page.
  • HTML markup can be included in the text.

Days In The New Table

  • The number of days that new products are listed in the New Items list.
  • This value should be given; there is no default value.

New Items List Masthead

  • Optional; if blank, the standard masthead used.
  • If given, overrides the standard masthead on the New Items page.
  • Enter the absolute path location from your Web directory root to the banner graphic.
  • Example: /images/newbanner.gif

Special Sale Items List Masthead

  • Optional; if blank, the standard masthead used.
  • If given, overrides the standard masthead on the Special Sale page.
  • Enter the absolute path location from your Web directory root to the banner graphic.
  • Example: /images/specbanner.gif

Zero Price Alternate Text Display

  • Optional; if blank, prices will be shown as $0.00.
  • If given, replaces the zero price with the given text.
  • This is overridden by the same field in the Category database.

New Product Description Template

  • Optional; full HTML can be placed in this field.
  • The contents of this field will be placed in the Product Description field of new products added to the online catalog. This will allow you to have a standard display template without having to re-enter it every time.
